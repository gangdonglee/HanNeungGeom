# 한능검 문제풀이 (HanNeungGeom)

한국사능력검정시험 기출문제를 웹에서 한 문제씩 풀어보는 모바일용 앱입니다.

## 실행

별도 빌드 없이 `index.html`을 브라우저로 열면 됩니다.

- **GitHub Pages**: https://gangdonglee.github.io/HanNeungGeom/

## 기능

- 한 문제씩 풀이 → 정답 확인 → 해설 → 다음
- 단원(시대)별 필터
- 오답노트 + 북마크
- 점수·결과 화면
- 문제 셔플(랜덤 출제)
- 이어풀기 저장 (localStorage)

## 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 앱 화면·로직 (바닐라 JS, 모바일 우선) |
| `questions.js` | 문제 데이터 |

## 데이터

정답·해설은 직접 검증하여 입력합니다. 지도·유물 등 이미지 자료가 필요한 문제는 별도 보강 예정입니다.

현재 수록: 73회 심화 일부 (검증 완료분).
